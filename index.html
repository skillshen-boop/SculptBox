Features

Brush Management

Thousands of brushes, millisecond response. Full thumbnail preview — no more guessing from file names.

Thumbnail Preview

Every brush shows its real thumbnail. Native ZBP parsing, 50ms per brush. S/M/L sizes for any workflow.

Search & Tags

Instant keyword search. 10 auto-tags (Culpting, Curve, Insert, Utility) plus custom tags.

Tag System

Auto-tags from file names + manual tagging. Click to filter. Persists across sessions.

Favorites

Star your most-used brushes. Access them in one click. Always at the top of your library.

S/M/L Grid Sizes

Switch between Small, Medium, and Large thumbnail sizes. Adapt the view to your workflow.

Send to ZBrush

Right-click send, real-time IPC bridge. Batch send multiple brushes. No more 300-brush limit.

Batch Operations

Batch send, batch export, batch tag. Built-in progress tracking. One-click bridge deployment.

Plugin Status

Health check dashboard. ZBrush status, bridge, IPC readiness at a glance. Install in 1 minute.

Settings & About

Full settings: scan directories, ZBrush path, cache, language, update, about, copyright.
