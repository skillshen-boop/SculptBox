## ⚠️ 首次安装

macOS 可能会提示 **"SculptBox 已损坏，无法打开"**。

这不是文件坏了。终端执行：

```bash
sudo xattr -rd com.apple.quarantine /Applications/SculptBox.app
```

或 **系统设置 → 隐私与安全性 → 仍然打开**。

---

## 🚀 快速开始

1. 下载 DMG，拖入 Applications
2. 如果报错，执行上面的命令
3. 启动 SculptBox
4. Settings → Plugin Status → Install 部署桥接
5. 重启 ZBrush
