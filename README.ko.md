## ⚠️ 첫 설치

macOS에서 **"SculptBox가 손상됨"** 메시지가 뜨면：

```bash
sudo xattr -rd com.apple.quarantine /Applications/SculptBox.app
```

또는 **시스템 설정 → 개인정보 보호 및 보안 → 계속 열기**.

---

## 🚀 빠른 시작

1. DMG 다운로드 후 Applications에 드래그
2. 차단되면 위 명령어 실행
3. SculptBox 실행
4. Settings → Install로 브릿지 배포
5. ZBrush 재시작

## ✨ 기능

- **브러시 관리** — 검색, 태그, 즐겨찾기, 분류
- **썸네일 미리보기** — 네이티브 ZBP 파싱
- **Send to ZBrush** — 우클릭으로 전송
- **배치 작업** — 일괄 전송, 일괄 내보내기
- **4개 언어** — English · 中文 · 日本語 · 한국어
- **원클릭 배포** — 1분 안에 설치 완료
