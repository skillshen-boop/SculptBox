## 🚀 快速开始

1. 下载 DMG，拖入 Applications
2. 启动 SculptBox
3. Settings → Plugin Status → Install 部署桥接
4. 重启 ZBrush

---

## ✨ 功能

- **笔刷管理** — 搜索、标签、收藏、分类，万级笔刷库毫秒级响应
- **缩略图预览** — 原生 ZBP 解析，所见即所得
- **Send to ZBrush** — 右键发送，实时加载
- **批量操作** — 批量发送、批量导出
- **4 语言** — 中文 / English / 日本語 / 한국어
- **一键部署** — 1 分钟完成桥接安装
